Comparaciones M√∫ltiples

Uso de comparaciones m√∫ltiples para evaluar las diferencias en las medias de grupo

La comparaci√≥n de las medias mediante m√∫ltiples pruebas permite analizar cu√°les medias son distintas y cuantificar el nivel de diferencia. Puede determinar la relevancia estad√≠stica de estas diferencias mediante un conjunto de intervalos de confianza, pruebas de hip√≥tesis o ambas herramientas. Adem√°s de la relevancia estad√≠stica, los intervalos de confianza le permiten evaluar la relevancia pr√°ctica de las diferencias entre medias. Como es habitual, si el intervalo de confianza no contiene el valor cero, se rechaza la hip√≥tesis nula que sostiene que no hay diferencia entre las medias.

El m√©todo de comparaciones m√∫ltiples es una t√©cnica estad√≠stica que se utiliza para comparar varias poblaciones o grupos en un estudio y detectar diferencias significativas entre ellos.

El m√©todo de comparaciones m√∫ltiples es √∫til cuando se realizan m√∫ltiples pruebas de hip√≥tesis y se desea controlar la tasa de error global (es decir, la probabilidad de cometer al menos un error de tipo I). Si se realiza una sola prueba, la probabilidad de cometer un error de tipo I (rechazar incorrectamente la hip√≥tesis nula) se puede controlar utilizando un nivel de significancia fijo. Sin embargo, cuando se realizan m√∫ltiples pruebas, la probabilidad de cometer al menos un error de tipo I aumenta a medida que se realizan m√°s pruebas.

Existen diferentes m√©todos de comparaciones m√∫ltiples, pero algunos de los m√°s comunes son:

  • Correcci√≥n de Bonferroni: Este m√©todo ajusta el nivel de significancia para controlar la tasa de error global, dividiendo el nivel de significancia por el n√∫mero de pruebas realizadas. Por ejemplo, si se realizan 5 pruebas con un nivel de significancia de 0.05, el nivel de significancia ajustado ser√≠a 0.05/5 = 0.01. \[\alpha_{corregido} = \frac{\alpha}{numero \ de \ grupos}\]

  • Correcci√≥n de Holm: Este m√©todo es una mejora del m√©todo de Bonferroni, que es m√°s poderoso y menos conservador. Primero, ordena los p-valores de menor a mayor. Luego, multiplica el p-valor m√°s peque√±o por el n√∫mero total de pruebas (m) y lo compara con el nivel de significancia. Si el p-valor es menor o igual que el nivel de significancia, se rechaza la hip√≥tesis nula. Si no, se pasa al siguiente p-valor y se multiplica por m-1, y as√≠ sucesivamente hasta que se encuentra un p-valor que no cumple con el criterio de rechazo.

  • FDR (tasa de descubrimiento falso): Este m√©todo ajusta los p-valores para controlar la tasa de descubrimiento falso, que es la proporci√≥n de hip√≥tesis nulas rechazadas que son verdaderas. El m√©todo FDR permite detectar m√°s diferencias significativas que los m√©todos de Bonferroni y Holm, pero a costa de una mayor tasa de falsos positivos.

En resumen, el m√©todo de comparaciones m√∫ltiples es una t√©cnica importante para controlar la tasa de error global en estudios con m√∫ltiples pruebas de hip√≥tesis y puede ayudar a identificar diferencias significativas entre m√∫ltiples poblaciones o grupos.

Bonferroni

Para calcular el valor p ajustado por Bonferroni, se debe seguir los siguientes pasos:

  1. Realizar las pruebas estad√≠sticas que se deseen y obtener el valor p para cada una de ellas.

  2. Multiplicar cada valor p obtenido por el n√∫mero total de pruebas realizadas.

  3. Comparar cada valor p ajustado con el nivel de significancia deseado (por ejemplo, 0,05) y considerar los resultados significativos solo si el valor p ajustado es menor o igual que el nivel de significancia.

  4. Reportar los resultados significativos junto con su valor p ajustado por Bonferroni.

A modo de ejemplo, supongamos que se realizan 3 pruebas y se obtienen los siguientes valores p: 0.02, 0.04 y 0.01. El n√∫mero total de pruebas es 3, por lo que se multiplica cada valor p por 3, obteniendo 0.06, 0.12 y 0.03, respectivamente. Si se desea un nivel de significancia de 0.05, se consideran significativos solo los resultados con valores p ajustados iguales o menores a 0.05. En este caso, el √∫nico resultado significativo ser√≠a el tercer valor p ajustado, que es 0,03.

Holm

El m√©todo de Holm tambi√©n se utiliza para controlar el n√∫mero de errores tipo I en comparaciones m√∫ltiples de hip√≥tesis. A continuaci√≥n, se presentan los pasos para calcular el umbral de significancia utilizando el m√©todo de Holm:

  1. Ordenar los valores de p (valor de probabilidad) para cada hip√≥tesis nula de menor a mayor.
  2. Para cada valor de p en la lista ordenada, multiplicar el valor de p por el n√∫mero de hip√≥tesis restantes en la lista (despu√©s de esa hip√≥tesis).
  3. Ordenar los valores obtenidos en el paso 2 de mayor a menor.
  4. Establecer un umbral de significancia como el valor m√°s peque√±o de p en la lista original para el cual el valor de p multiplicado por el n√∫mero de hip√≥tesis restantes en la lista es menor que el n√∫mero de hip√≥tesis.

La idea detr√°s de la correcci√≥n de Holm es fingir que est√°s haciendo las pruebas secuencialmente; comenzando con el valor p m√°s peque√±o (en bruto) y pasando al m√°s grande. Para el j-√©simo mayor de los valores p, el ajuste es

\[p^{\prime}_j = \max \{j\times p_j, p^{\prime}_{j+1}\}\] donde \(j=n-i+1\) y \(i\) es el ranking de \(p_i\).

Primero, ordena todos sus valores p en orden, de menor a mayor. Para el valor p m√°s peque√±o, todo lo que tiene que hacer es multiplicarlo por m y listo. Sin embargo, para todos los dem√°s es un proceso de dos etapas. Por ejemplo, cuando pasa al segundo valor p m√°s peque√±o, primero lo multiplica por m‚àí1. Si esto produce un n√∫mero que es mayor que el valor p ajustado que obtuvo la √∫ltima vez, entonces lo conserva. Pero si es m√°s peque√±o que el √∫ltimo, copia el √∫ltimo valor p.¬†Para ilustrar c√≥mo funciona esto, considere la siguiente tabla.

FDR

El ajuste del valor p por tasa de descubrimiento falso (FDR, por sus siglas en ingl√©s) es un m√©todo de correcci√≥n para controlar la tasa de error tipo I en pruebas m√∫ltiples, sin ser tan conservador como el m√©todo de Bonferroni. El FDR se refiere a la proporci√≥n de resultados significativos que se espera que sean falsos positivos.

Para calcular el valor p ajustado por FDR, se deben seguir los siguientes pasos:

Realizar las pruebas estad√≠sticas que se deseen y obtener el valor p para cada una de ellas.

Ordenar los valores p obtenidos de menor a mayor.

Calcular la tasa de descubrimiento falso (FDR) para cada valor p, utilizando la siguiente f√≥rmula:

\[FDR(p) = (p \times m)/k\]

Donde \(p\) es el p-valor para una muestra determinada, \(m\) es el n√∫mero total de pruebas y \(k\) es el rango de la prueba (es decir, la posici√≥n en el orden de los valores p).

  1. Establecer un umbral de FDR deseado (por ejemplo, 0.05) y encontrar el valor p correspondiente a ese umbral de FDR.

  2. Todos los valores p por debajo del umbral de FDR deseado se consideran significativos.

A modo de ejemplo, supongamos que se realizan 5 pruebas y se obtienen los siguientes valores p: [0.03, 0.02, 0.05, 0.01, 0.07]

  1. Se obtienen los valores p: 0.03, 0.02, 0.05, 0.01, 0.07.

  2. Se ordenan de menor a mayor: 0.01, 0.02, 0.03, 0.05, 0.07.

  3. Se calcula la tasa de descubrimiento falso para cada valor p, utilizando la f√≥rmula FDR(p) = (p * m) / k, donde m es el n√∫mero total de pruebas y k es el rango de la prueba:

FDR(0.01) = (0.01 * 5) / 1 = 0.05

FDR(0.02) = (0.02 * 5) / 2 = 0.05

FDR(0.03) = (0.03 * 5) / 3 = 0.05

FDR(0.05) = (0.05 * 5) / 4 = 0.0625

FDR(0.07) = (0.07 * 5) / 5 = 0.07

  1. Se establece el umbral de FDR deseado. Por ejemplo, si se desea un umbral de 0.05, se selecciona el √∫ltimo valor p con una tasa de descubrimiento falso menor o igual a 0.05. En este caso, ese valor es 0.03.

  2. Los valores p correspondientes a las pruebas con una tasa de descubrimiento falso menor o igual al umbral seleccionado se consideran significativos. En este caso, los valores p correspondientes a las pruebas H4, H2 y H1 son significativos, ya que tienen una tasa de descubrimiento falso menor o igual a 0.05. Los valores p correspondientes a las pruebas H3 y H5 no son significativos, ya que su tasa de descubrimiento falso es mayor a 0.05.

Es importante tener en cuenta que el m√©todo FDR no garantiza el control de errores tipo I a nivel individual para cada hip√≥tesis nula, sino que controla el n√∫mero esperado de falsos descubrimientos en el conjunto de hip√≥tesis nulas rechazadas.
